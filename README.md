# 직방 데이터 스크래퍼 Chrome 확장 프로그램

이 Chrome 확장 프로그램은 직방 웹사이트에서 매물 정보를 자동으로 수집하고, 수집된 데이터를 CSV 파일로 다운로드할 수 있도록 도와줍니다. 특정 기간(2026년 1월 ~ 3월) 내의 매물 등록 개수를 팝업에서 바로 확인할 수 있습니다.

## ✨ 주요 기능

-   **매물 정보 수집**: 직방 웹사이트에서 원룸, 오피스텔(도시형생활주택), 빌라 카테고리의 매물 정보를 수집합니다.
-   **CSV 다운로드**: 수집된 모든 데이터를 CSV 파일로 자동 다운로드합니다.
-   **기간별 요약**: 2026년 1월부터 3월까지 등록된 매물의 개수를 팝업에서 즉시 확인합니다.
-   **진행 상태 표시**: 데이터 수집 중 팝업에서 진행 상태를 시각적으로 보여줍니다.

## 🚀 설치 방법

1.  **코드 다운로드**:
    *   이 GitHub 저장소 페이지 ([https://github.com/eunsu-ko/zigbang_scraper_extension](https://github.com/eunsu-ko/zigbang_scraper_extension)) 에 접속합니다.
    *   페이지 상단에 있는 초록색 **`< > Code`** 버튼을 클릭하고 **`Download ZIP`**을 선택하여 파일을 다운로드합니다.
    *   다운로드한 `zigbang_scraper_extension-main.zip` 파일의 압축을 풉니다.
2.  **Chrome 확장 프로그램에 로드**:
    *   Chrome 브라우저를 열고, 주소창에 `chrome://extensions` 를 입력 후 Enter 키를 누릅니다.
    *   페이지 오른쪽 상단의 **'개발자 모드(Developer mode)'** 토글을 켜 활성화합니다.
    *   왼쪽 상단에 나타나는 **'압축해제된 확장 프로그램을 로드합니다(Load unpacked)'** 버튼을 클릭합니다.
    *   파일 선택 창에서 압축을 푼 폴더(`zigbang_scraper_extension-main`)를 선택한 후 '폴더 선택'을 클릭합니다. (`manifest.json` 파일이 직접 포함된 폴더를 선택해야 합니다.)
3.  **확인**: 이제 '직방 데이터 스크래퍼'가 확장 프로그램 목록에 추가된 것을 볼 수 있습니다.

## 💡 사용 방법

1.  Chrome 브라우저에서 직방 웹사이트((https://ceo.zigbang.com/ads/oneroom))로 이동합니다.
2.  브라우저 우측 상단의 **퍼즐 모양(확장 프로그램) 아이콘**을 클릭합니다.
3.  목록에서 **'직방 데이터 스크래퍼'**를 찾아 클릭하여 팝업을 엽니다.
4.  팝업 창에서 **'데이터 수집 시작'** 버튼을 클릭합니다.
5.  데이터 수집이 진행되는 동안 팝업 창에 진행 상태 메시지("수집 중...")가 표시됩니다.
6.  수집이 완료되면, 수집된 데이터를 담은 CSV 파일이 자동으로 다운로드됩니다.
7.  팝업 창에는 "Y26 Q1 직방 등록 매물 수 XX개 (오늘날짜 YYYY-MM-DD 기준)"와 같은 형식으로 최종 수집 개수가 표시됩니다.
